## Installation

- clone the project inside your local develepment for php
- change the  FILESYSTEM_DISK=local to public
- create the database in your pc called . laravel
- run composer install
- php artisan storage:link
- php artisan migrate


## Requirements

- Mysql Databaase
- php
- php composer
